Data

This produces a file with extension .fms containing a dump of the data that is sent to FOMUS before any processing occurs. If this is the only output specified, FOMUS simply saves the data and returns without further processing. It may be specified using either the :DATA or :FOMUS keywords in the OUTPUT setting. The file is editable and can be reloaded for editing and processing by calling the FOMUS function with the name of the file. See Text File Interface for a description of this file format.
